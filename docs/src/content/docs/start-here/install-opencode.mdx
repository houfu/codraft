[OpenCode](https://opencode.ai) is a third-party client compatible with Claude Code plugins. Codraft uses the standard Claude Code plugin format, so it works with any compatible client -- including OpenCode.

## Plugin details

| Field | Value |
|-------|-------|
| Plugin identifier | `houfu/codraft` |
| Repository | [github.com/houfu/codraft](https://github.com/houfu/codraft) |

## Installation

Refer to [OpenCode's documentation](https://opencode.ai) for the specific plugin installation method. The general approach is the same as Claude Code: add the `houfu/codraft` plugin via the marketplace or plugin management command.

<Aside type="caution">
OpenCode is a third-party project and is not maintained by the Codraft team. Plugin compatibility and behaviour may vary from the official Claude Code experience. If you encounter issues specific to OpenCode, check their documentation or issue tracker first.
</Aside>
